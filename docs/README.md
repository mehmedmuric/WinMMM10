## Quick Start

1. **Installation**: Run the installer or build from source
2. **Create Project**: File → New Project
3. **Load Binary**: File → Load Binary
4. **Detect Maps**: Map → Detect Maps
5. **Edit**: Select maps and edit in viewers
6. **Save**: File → Save Binary

## Features

- ✅ Binary file editing (up to 16MB+)
- ✅ HEX editor with undo/redo
- ✅ Automatic map detection
- ✅ 2D/3D map visualization
- ✅ Map packs (import/export)
- ✅ Multiple checksum algorithms
- ✅ Plugin system
- ✅ Project management
- ✅ Dark theme UI
